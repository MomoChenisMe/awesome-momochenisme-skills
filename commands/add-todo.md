# Add Todo

Add a new task to the todo list.

## Usage
/add-todo <task description>

## Instructions
1. Read `TODOS.md` from workspace root
2. Append `- [ ] $ARGUMENTS` under the `# 待辦事項 (Active Todos)` section
3. Write the updated file
4. Confirm the addition

## Example
```
/add-todo 買牛奶
```
Adds `- [ ] 買牛奶` to the active todos list.
