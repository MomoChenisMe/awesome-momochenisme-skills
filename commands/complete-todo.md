# Complete Todo

Mark a todo item as completed.

## Usage
/complete-todo <task keyword or number>

## Instructions
1. Read `TODOS.md` from workspace root
2. Find the matching task in `# 待辦事項 (Active Todos)`
3. Remove it from active todos
4. Add to `# 已完成事項 (Completed Todos)` as:
   `- [x] ~~task text~~ (完成: YYYY-MM-DD)`
5. Write the updated file
6. Confirm completion

## Example
```
/complete-todo 牛奶
```
Finds the task containing "牛奶" and marks it complete.
