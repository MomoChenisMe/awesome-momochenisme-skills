# Delete Todo

Remove a todo item from the list.

## Usage
/delete-todo <task keyword or number>

## Instructions
1. Read `TODOS.md` from workspace root
2. Find the matching task (search in both active and completed)
3. Remove the entire line
4. Write the updated file
5. Confirm deletion

## Example
```
/delete-todo 牛奶
```
Removes the task containing "牛奶".
