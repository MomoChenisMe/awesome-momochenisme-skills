# Sync Scribe (簡短繁體中文版)

專為 Claude Code 設計的簡化筆記與待辦事項管理插件。

## 功能特色

- 🎯 **自動分類**：自動判斷輸入是待辦事項（Todo）還是記事（Note）。
- ✅ **待辦管理**：新增、完成與刪除待辦事項，無需繁瑣的時間設定。
- 📝 **分類記事**：自動將想法歸類至「工作」、「生活」或「未分類」。
- 🔍 **簡單直觀**：所有資料儲存在本地的 `TODOS.md` 與 `NOTES.md`。

## 安裝方式

### Claude Code Plugin
```bash
# 本地開發測試
claude --plugin-dir ./sync-scribe-plugin
```

## 使用方式

### 斜線指令 (Commands)
| 指令 | 說明 |
|------|------|
| `/add-todo <任務內容>` | 新增一個待辦事項 |
| `/complete-todo <關鍵字>` | 標記該事項為已完成 |
| `/list-todos` | 列出所有待辦事項 |
| `/delete-todo <關鍵字>` | 刪除待辦事項 |
| `/add-note <記事內容>` | 新增一則筆記（自動分類） |
| `/list-notes [分類]` | 列出筆記（可依分類篩選） |
| `/delete-note <關鍵字>` | 刪除筆記 |

### 自然語言 (Skill)
直接對 Claude 說：
- 「幫我記得買咖啡」 -> 自動加入待辦
- 「記一下：這杯咖啡很好喝」 -> 自動加入筆記
- 「買咖啡完成了」 -> 自動標記完成

## 檔案結構
- `TODOS.md`：存放待辦事項。
- `NOTES.md`：存放分類記事。
